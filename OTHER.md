# Platforms

It depends on what operating system what you are using.

* Windows
You can use **setup-powershell.ps1** to build the app, or **setup.bat** too.

Otherwise, you can write the commands yourself.

```bash
python build.py
python main.py
```

* Linux/MacOS

You can run **setup-linux.sh** for Linux

You need to use these commands for MacOS:

```bash
python3 build.py
python3 main.py
```

* ChromeOS

You need to enable **Linux development environment** and run **setup.sh**
